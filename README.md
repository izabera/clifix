# clifix

That nifty autocorrection feature in zsh is now available for bash!

![correction system](http://i.imgur.com/798lKDc.png)


## Requirements
- bash 4.0 or above
- nothing else!

## How to install
- clone this repo
- run `./install`
- add `source /path/to/clifix` in your `.bashrc`
